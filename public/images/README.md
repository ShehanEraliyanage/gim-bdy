# Photos for Birthday Carousel

Add your birthday celebration photos here. Supported formats:
- JPG / JPEG
- PNG
- WebP

The app will automatically load images from this folder and display them in a 3D carousel after the gift is unwrapped.

## How to add photos:

1. Place your favorite birthday photos in this folder
2. The photos will be displayed in the carousel by their filename alphabetically
3. You can navigate through them with:
   - Arrow buttons (← →)
   - Arrow keys on keyboard
   - Thumbnail selector

## Photo Tips:

- Recommended size: 1080x1080px (square format works best)
- Keep file sizes reasonable (< 2MB each) for faster loading
- Use 5-10 photos for the best experience
